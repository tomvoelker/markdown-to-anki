Markdown to Anki

Built for the annoying gap between generated markdown and what Anki actually accepts.

Convert markdown plus LaTeX into paste-ready Anki HTML. Single cards and batch blocks both stay local, auto-copy the result, and give you a quick preview before you paste.

single card batch mode local-only processing

What it handles

▾
  • Formatting: turns markdown emphasis into Anki-friendly HTML.
  • Math: supports inline and block LaTeX, plus common environments.
  • Lists: converts bullet and numbered lists into clean HTML.
  • Workflow: copies converted output automatically as you type.

Prompt snippet

append this to an LLM prompt to get paste-ready output ▾ 
Format your response as Anki flashcard(s) using the following rules:

- Use **double asterisks** for bold and *single asterisks* for italic.
- For inline math, write \(expression\) — e.g. \(x^2 + y^2 = r^2\)
- For block/display math, write \[expression\] on its own line.
- For lists, use - (bullet) or 1. 2. 3. (numbered) in standard Markdown.
- Do NOT output raw HTML, , or any other markup.
- Do NOT use $ or $$ for math — only \( \) and \[ \].

For a single card, output:
Question text here

Answer text here

For multiple cards (batch mode), wrap each question and answer in triple backticks and separate cards with five dashes:
```
Question text here
```

```
Answer text here
```

-----

```
Next question here
```

```
Next answer here
```
